## 第二十部分：机要通信（Confidential Communications）特殊政治红线与绝密件处理政策

> 说明：本文件保留制度层级与业务边界，但凡涉及具体环保限值、仓储周期、无主件收益处置、隐私留痕时长等内容，均应以现行法规、地方执行口径和平台公告为准。

中国邮政核心保密网络（机要通信渠道）专门承运党政军领率机关的绝密、机密、秘密文件和军事档案，在机器人知识库中执行独立的安全及隔离政策。

### 20.1 机要通信绝对隔离与准入红线

- **主体资质绝对受限：** 机要通信渠道**绝对不接受任何个人、私营企业、外资企业的交寄申请**。其服务对象仅限于县级以上党政机关、军队团级以上单位、部分特定国有企事业单位。机器人对任何个人触发的“寄送机要文件”意图必须实行刚性拒绝并话术拦截。
- **交寄物理隔离：** 机要邮件不通过普通邮政网点收寄，亦不进入普通快递（EMS）分拣中心。必须由寄件单位指定的、持有特殊机要交寄证件的机要员，前往各省市专门设立的“机要通信留存点”或特设窗口当面交寄。

### 20.2 机要邮件在途安全与中途处置政策

- **严禁任何形式的开拆验视：** 机要邮件适用远高于普通寄递的保密规则，不进入普通验视流程。其开拆、接触和移交流程应受专门保密制度约束。
- **物理签收要求：** 机要邮件末端交接通常只允许在具备相应保密职责的接收主体之间完成，不适用普通快递式代签流程。

## 第二十一部分：绿色物流与生态环保：包装物料资质、限制过度包装与有毒有害限值政策

根据国家邮政局《快递包装重金属与特定物质限量》及《邮政业寄递安全监督管理办法》，中国邮政对收寄、打包环节的物料实施刚性环保合规审计数据指标。

### 21.1 协议客户自带包装物料及有害物质红线数据

大客户、电商卖家自行采购并打包的纸箱、塑料袋、气泡膜，进入中国邮政收寄网络时，其物料材质必须 100% 符合以下数据限量：

| 有害物质/物料指标分类 | 刚性限值及检验标准数据 |
| --- | --- |
| **铅（Pb）及化合物限量** | 需符合现行快递包装环保标准和有害物质限量要求。 |
| **汞（Hg）及化合物限量** | 需符合现行快递包装环保标准和有害物质限量要求。 |
| **镉（Cd）及化合物限量** | 需符合现行快递包装环保标准和有害物质限量要求。 |
| **苯系物（溶剂残留）** | 应满足包装材料安全与挥发性物质控制要求。 |
| **重金属总量叠加限值** | 应满足国家现行包装环保标准。 |

### 21.2 限制过度包装与胶带缠绕规范政策

- **二次包装强制禁止：** 凡是商品自带的外包装箱已经具备足够防震、抗压、防水功能，且符合长途机械分拣强度的（如原厂直发小家电、原装纸箱牛排等），中国邮政营业网点**严禁对其进行二次纸箱加装包装**，必须直接黏贴电子面单发运。
- **胶带一字/十字缠绕红线：** 对普通纸箱封箱时，严禁实施全包裹、多层层叠、木乃伊式缠绕。
- 箱体较小的普通纸箱，通常以简化封箱为主，避免过度缠绕。
- 中大规格纸箱则更强调受力均衡和封装牢固，但仍应避免明显过度包装。

## 第二十二部分：国内无法投递又无法退回邮件（国内无主件）的甄别、开拆与变卖政策细则

依据《中华人民共和国邮政法实施细则》第四章，针对国内普通包裹及快递，触发“双向无法投递”节点后的刚性处置政策。

### 22.1 国内无主邮件（Dead Letter）的认定周期数据

- **第一阶段（退回失败）：** 邮件因地址错误无人领收，退回原寄地时，若寄件人也搬迁、信息虚假、或明确声明拒收退件，邮件当天转入**国内无主邮件暂存库**。
- **第二阶段（法定保留期）：** 自转入无主库之日起，中国邮政分公司对该类邮件执行 **6 个月** 的刚性无条件物理保留保管期。期间产生的仓储费用计入邮件账单。

### 22.2 无主邮件的法定开拆、清理与收益分配政策

- **监督开拆：** 6 个月保管期满仍无人认领的，无主邮件开拆、清理和处置通常需要在合规监督和内部审计框架下进行，不适用普通随意处置流程。
- **内件分类分流执行政策：**

```text
                         【6个月无主邮件开拆】
                                   │
         ┌─────────────────────────┼─────────────────────────┐
   [现金/有价证券]            [一般商品/有价值物资]       [个人隐私文件/照片]
         │                         │                         │
   (全额上缴国库)            (依法委托拍卖/变卖)         (实施强制化学销毁)
         │                         │                         │
   【财政部专项账户】         【扣除仓储费后上缴】       【防止任何信息泄露】
```

- **拍卖收益红线规定：** 无主邮件变卖所得通常需要按照法定处置流程处理，不得由企业或个人私自截留、私分或私下购买。

## 第二十三部分：跨境集运业务（中邮集运）免仓期、逾期仓储费与自动弃货判定政策

中邮海外购及中国邮政针对境外买家、跨境留学生设立的国内集运仓（Consolidation Warehousing），执行高周转率的容积控制与逾期清理政策。

### 23.1 集运仓仓储服务费及免费期数据

- **基础免仓期：** 集运仓通常设有免费仓储期，但具体时长因平台、仓库和时期不同而变化。
- **逾期计费红线数据：**
- 超出免费期后，平台通常会按日或按件收取仓储费用。
- **计费标准：** 小件与大件包裹的仓储费往往不同，具体以当期仓储规则为准。
- 体积较大或重量较高的包裹，通常适用更高的仓储费率。

### 23.2 自动弃货（Automatic Abandonment）判定红线

- **超期判定：** 包裹在仓内停留过久且长期未绑定、未出库、欠费较高时，平台可能将其认定为超期件或弃货候选件。
- **系统处置：** 一旦进入超期清理流程，平台可能执行销毁、清货或其他处置措施，后续确权空间会显著缩小。

## 第二十四部分：用户个人信息保护、面单脱敏及隐私安全合规边界政策

依据《中华人民共和国个人信息保护法》及邮政行业安全标准，中国邮政全网系统（含客服机器人）在处理、回传用户轨迹及隐私数据时，执行最严格的物理与逻辑边界。

### 24.1 电子面单个人信息模糊化（脱敏）标准数据

进入中国邮政全自动化分拣、投递终端系统的面单及电子数据流，必须强制执行以下字段脱敏算法：

- **姓名（Name）脱敏：** 通常仅展示部分识别信息，其余字符以星号或其他方式模糊处理。
- **电话（Phone Number）脱敏：** 国内手机号固定模糊化中间 4 位。格式严格执行 **138****1234**。
- **地址脱敏：** 打印在物理面单上的收件地址通常会隐藏部分精确定位信息，详细门牌信息一般仅对具备权限的末端投递环节开放。

### 24.2 客服机器人/人工通道隐私查询限制

- **非主体查询拦截：** 当查询人手机号、姓名与中国邮政内网核心面单系统注册的“寄件人”或“收件人”数据不一致时，在线机器人及人工客服**绝对拒绝提供除宏观物流轨迹（如：已到达北京中心局）之外的任何隐私数据**（包含不得透露具体收件人电话、具体收发地址、代签收人手写签名图片等）。
- **内网日志留痕：** 调取隐私面单数据通常会保留访问日志、接口标识和设备痕迹，以满足审计和安全追溯要求。
