# China Post Internship Documentation

This documentation site corresponds to the reconstructed version of the China Post internship project.

The main project work was already completed on the company machine. The current repository is a rebuilt and reorganized version based on the finished structure, because the original environment and part of the data could not be moved out in full.

This site keeps three things clear:

1. The main project phases.
2. Why the current repository is presented as a rebuilt version.
3. The final summary kept for the model fine-tuning work.
