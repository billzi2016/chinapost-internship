# 邮政实习项目文档站点

这个文档站点对应的是邮政实习项目的整理版资料。

项目主体当时已经在公司电脑上完成，当前仓库里的代码、说明和目录结构，是在原始环境和部分数据无法完整迁出的情况下，按已经完成的结果重新整理出来的重建版本。

当前站点主要保留三部分信息：

1. 项目主线和阶段划分。
2. 当前仓库为什么会呈现为“重建整理版”。
3. 模型微调和系统方向里最终保留的结论口径。

如果只看当前仓库，最重要的理解方式是：

1. `week1`、`week2`、`week3` 更接近任务阶段，不严格等于自然周。
2. `docs-site` 是按原先已完成结构重新整理出来的文档工程入口。
3. 当前文档以可复现、可说明、可继续补充为主，不再按公司内网环境原样展开。
