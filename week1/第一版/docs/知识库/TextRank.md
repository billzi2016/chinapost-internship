# TextRank 方法说明

## 1. 方法定位

`TextRank` 是本项目里第二种关键词提取方法。  
它和 `TF-IDF` 的区别在于：

- `TF-IDF` 更偏统计频率
- `TextRank` 更偏词与词之间的关系结构

对应代码位置：

- `/Users/bizi/Desktop/邮政实习/week1/stats/textrank_analysis.py`

## 2. 原理

`TextRank` 的核心思想来自 `PageRank`。  
`PageRank` 原本是网页排序算法，它假设：

- 一个网页如果被很多重要网页指向
- 那它自己也更重要

在文本里，这个思想被改写成：

- 一个词如果经常和很多重要词共同出现
- 那它本身也可能是重要词

因此，`TextRank` 的步骤通常是：

1. 构建词共现图
2. 图中的每个词是一个节点
3. 经常一起出现的词之间连边
4. 在这个图上做迭代打分
5. 分数高的词就是关键词

## 3. 本项目里具体怎么做

### 3.1 仍然使用“单条对话”为分析单元

本项目先从基础统计模块拿到 `documents`。  
每个 `document` 都是：

- 一条完整对话对应的 token 序列

和 `TF-IDF` 一样，`TextRank` 也不是按单轮 utterance 分析，而是按整条对话分析。

### 3.2 先过滤 token

每条文档在进入图构建之前，都会先走：

- `keyword_utils.filter_document`

这样做的原因和 `TF-IDF` 一样：

- 中文客服语料里有很多无效高频词
- 如果不过滤，共现图会被噪声淹没

### 3.3 如何构建共现图

项目里设定了一个共现窗口：

- `WINDOW_SIZE = 4`

具体做法是：

- 遍历文档里的每个词
- 看它后面窗口范围内的词
- 如果两个词在这个局部窗口内共同出现，就建立连接

这意味着：

- 图中的边不是靠句法分析得到的
- 而是靠“局部相邻共现”得到的

这种方式简单，但对关键词提取很常见也很有效。

### 3.4 图结构是什么样

项目里使用的是：

- 无向图
- 邻接集合形式

也就是说，如果 `A` 和 `B` 共现，就会同时记录：

- `A -> B`
- `B -> A`

### 3.5 如何迭代打分

项目里使用的是 PageRank 风格迭代：

- 阻尼系数 `DAMPING = 0.85`
- 迭代轮数 `ITERATIONS = 30`

每一轮会更新每个词的分数。  
一个词的新分数取决于：

- 与它相连的其他词当前分数有多高
- 那些词把多少权重“分给”它

直觉上：

- 如果一个词连接到很多重要词
- 它自己也会越来越重要

### 3.6 最后如何得到关键词

图迭代结束后，会按分数从高到低排序，取前 `top_k` 个词。  
另外，项目里还会补统计这个词在语料里出现了多少次，用于后续展示。

所以每个输出项包含：

- `token`
- `score`
- `count`

## 4. 为什么这个方法适合本项目

### 4.1 它不只看词频

在客服语料中，有些词单纯出现很多次，不代表它真的有主题意义。  
`TextRank` 会额外利用“这个词和哪些词一起出现”来判断重要性。

### 4.2 对主题词网络更敏感

如果某些业务词经常一起出现，比如：

- 订单
- 发票
- 退款
- 申请

那么它们之间更容易形成一个高权重共现子图。  
这时 `TextRank` 往往会比纯词频方法更容易把这类词顶出来。

### 4.3 不依赖外部大模型

它仍然是传统方法，不需要加载语义模型，  
所以实现轻量、跑起来稳。

## 5. 局限

### 5.1 仍然不是真正的语义理解

虽然它比 `TF-IDF` 更看关系，但本质上仍然没有真正理解语义。  
它只是通过共现结构去间接近似主题重要性。

### 5.2 对窗口和过滤规则敏感

如果窗口太大：

- 图会变得很密

如果窗口太小：

- 很多真实关联可能连不上

同时，如果 token 过滤做得不好，噪声词仍然会进入图。

### 5.3 结果解释比 TF-IDF 略复杂

`TF-IDF` 可以直说“这个词在局部高频、全局不泛滥”。  
`TextRank` 则需要解释：

- 它在共现网络里处于核心位置

虽然也能解释，但没有 `TF-IDF` 那么直观。

## 6. 在本项目中的定位

在当前项目里，`TextRank` 的最佳定位是：

- 作为 `TF-IDF` 的补充
- 用关系网络视角重新看关键词
- 帮助判断某些业务词是否形成稳定主题结构

它不是要替代 `TF-IDF`，而是帮助我们从另一个角度理解语料。
